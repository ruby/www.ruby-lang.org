Ruby 3.3.12 has been released.

This release includes security fixes.
Please check the topics below for details.

* [CVE-2026-41316: ERB @\_init deserialization guard bypass via def\_module / def\_method / def\_class]({% link en/news/_posts/2026-04-21-erb-cve-2026-41316.md %})

This release updates the default gem erb to 4.0.3.1 and the bundled gem net-imap to 0.4.25. The net-imap update fixes CVE-2026-42245, CVE-2026-42246, CVE-2026-42256, CVE-2026-42257, CVE-2026-42258, CVE-2026-47240, CVE-2026-47241 and CVE-2026-47242. Please see the release notes of [net-imap v0.4.24](https://github.com/ruby/net-imap/releases/tag/v0.4.24) and [net-imap v0.4.25](https://github.com/ruby/net-imap/releases/tag/v0.4.25) for the details of these fixes.

Please see the [GitHub releases](https://github.com/ruby/ruby/releases/tag/v3_3_12) for further details.
