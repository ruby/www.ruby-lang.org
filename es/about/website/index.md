Este sitio web fue generado con Ruby usando [Jekyll][jekyll],<br>
su código fuente está alojado en [GitHub][github-repo].

## Diseño

El diseño visual actual es de [Taeko Akatsuka][akatsuka].
El sitio fue renovado en diciembre de 2025.

Diseño visual anterior de [Jason Zimdars][jzimdars].<br>
Basado en un diseño previo del Ruby Visual Identity Team.

<img src="../../../images/about/screenshot-ruby-lang-old.png" alt="Diseño anterior de ruby-lang.org" width="400" style="border: 1px solid #d6d3d1;" />

## Logotipo

El [logotipo de Ruby][logo] tiene Copyright &copy; 2006, Yukihiro Matsumoto.


## Reportar problemas ##

Para reportar un problema usa el [issue tracker][github-issues]
o contacta a nuestro [webmaster][webmaster] (en inglés).


## Cómo contribuir ##

Este sitio web es mantenido con orgullo por miembros de la comunidad de Ruby.

Si deseas contribuir, lee las [instrucciones de contribución][github-wiki]
y simplemente comienza a abrir issues o pull requests.
