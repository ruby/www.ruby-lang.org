Ruby 3.3.12 がリリースされました。

このリリースにはセキュリティ修正が含まれています。詳細は以下の記事を参照してください。

* [CVE-2026-41316: ERB @\_init deserialization guard bypass via def\_module / def\_method / def\_class]({% link en/news/_posts/2026-04-21-erb-cve-2026-41316.md %})

このリリースでは default gem の erb を 4.0.3.1 に、bundled gem の net-imap を 0.4.25 に更新しています。net-imap の更新には CVE-2026-42245、CVE-2026-42246、CVE-2026-42256、CVE-2026-42257、CVE-2026-42258、CVE-2026-47240、CVE-2026-47241 および CVE-2026-47242 の修正が含まれます。これらの修正の詳細は [net-imap v0.4.24](https://github.com/ruby/net-imap/releases/tag/v0.4.24) および [net-imap v0.4.25](https://github.com/ruby/net-imap/releases/tag/v0.4.25) のリリースノートを参照してください。

詳細は [GitHub releases](https://github.com/ruby/ruby/releases/tag/v3_3_12) を参照してください。
