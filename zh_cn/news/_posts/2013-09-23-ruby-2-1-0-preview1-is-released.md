## 变动

值得提及的变动如下：

* VM (method cache)
* RGenGC
* refinements
* syntax
* Decimal Literal
* Frozen String Literal
* def's return value
* Bignum
* 128bit
* GMP
* String#scrub
* Socket.getifaddrs
* new RubyGems
