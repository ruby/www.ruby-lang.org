Ruby 1.8.7-p302 has been released([\[ruby-talk:367769\]][1]). This is a
security fix for CVE-2010-0541. See [the vulnerability report][2] for
more details.

I recommend all Ruby 1.8.7 users to upgrade the 1.8.7 to p302.
