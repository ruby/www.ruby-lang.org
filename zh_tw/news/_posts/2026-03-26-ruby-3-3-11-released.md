Ruby 3.3.11 已經發布了。 本次發布包含  [解決 zlib gem CVE-2026-27820 的更新](https://www.ruby-lang.org/zh_tw/news/2026/03/05/buffer-overflow-zlib-cve-2026-27820/)，以及其他錯誤修復。

詳細的變動請參閱 [GitHub 發布](https://github.com/ruby/ruby/releases/tag/v3_3_11)。

此為 Ruby 3.3 系列最後的常規維護版本。此版本發布後，Ruby 3.3 將進入安全維護階段。在此階段，我們將僅移植安全修復和關鍵建置問題的修復，不再進行任何常規的錯誤修復。

安全維護階段規劃為期一年，直至 2027 年 3 月底。屆時將終止 Ruby 3.3 的官方支援。我們建議您開始計劃升級至 Ruby 3.4 或 4.0。
