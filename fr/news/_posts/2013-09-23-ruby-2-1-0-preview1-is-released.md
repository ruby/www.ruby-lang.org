## Modifications

Les modifications les plus notables sont :

* VM (method cache)
* RGenGC
* refinements
* syntax
* Decimal Literal
* Frozen String Literal
* def's return value
* Bignum
* 128bit
* GMP
* String#scrub
* Socket.getifaddrs
* new RubyGems
