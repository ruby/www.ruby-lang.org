Найновіший онлайн ресурс для серйозної інформації про Ruby запущено.
[Ruby Code &amp; Style][1], онлайн журнал від [Artima][2], щойно
опублікував випуск #1. Перевірте імена в консультативній раді. Це
Хто є Хто з усіх, хто є кимось у світі Ruby. Прем'єрний
випуск має три видатні статті: Спочатку Jack Herrington, автор
Code Generation in Action (Manning, 2002) та Podcasting Hacks
(O'Reilly, 2005), написав [Модульні архітектури з Ruby][3] Далі,
Austin Ziegler дає нам [Створення друкованих документів з Ruby][4] І
є перевидання статті Ara Howard, [Кластеризація Linux з Ruby
Queue: Маленьке - це красиво][5], яка вперше з'явилася в Linux Journal, але
заслуговує повторної уваги Велике дякую консультативній раді, та
особливо Bill Venners за початок всього цього.
